# GymTrack

GymTrack é um aplicativo mobile híbrido desenvolvido com React Native, Expo e Firebase Firestore para acompanhamento de desempenho em treinos de musculação.

O aplicativo permite registrar exercícios, cargas e repetições, armazenando os dados em nuvem através do Firestore. Além disso, o sistema calcula automaticamente o volume de carga movimentado em cada exercício, auxiliando no monitoramento da progressão de treino.

O projeto foi desenvolvido como MVP com foco em persistência de dados, organização visual e experiência de uso simples e intuitiva.
